# Integration test host for indy-charts

This VitePress website is solely for the purpose of testing an actual implementation of the `@facioquo/indy-charts` library package to depict per-use charts in a realistic scenario.

It solely depends on the [`@facioquo/indy-charts`](../../libs/indy-charts/) package and indirectly on the [REST API](../../server/).  It must not directly depend on `@facioquo/chartjs-chart-financial` or any aspects of our primary client website.
